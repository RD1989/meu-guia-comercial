# Meu Guia Comercial SaaS - Profissional & IA

Plataforma SaaS avançada para guias comerciais municipais, com automação completa de conteúdo via Inteligência Artificial.

## 🚀 Funcionalidades Principais

- **Modelo SaaS Centralizado:** Gestão total por Super Admin.
- **Automação de Blog com IA:** Integração com OpenRouter (GPT-4o, Claude 3.5).
- **Banco de Referências de Conteúdo:** Colete fontes externas para a IA criar artigos.
- **Biblioteca de Mídias:** Gerencie imagens de apoio para postagens automáticas.
- **Design Premium:** Interface dark mode, responsiva e otimizada para conversão.
- **Vercel Ready:** Configurado para deploy imediato como SPA.

## 🛠️ Tecnologias

- **Frontend:** React + Vite + TypeScript
- **Styling:** Tailwind CSS + Shadcn UI
- **Backend:** Supabase (Auth, DB, Functions)
- **IA:** OpenRouter API

## 📦 Deploy na Vercel

1. Importe o repositório na Vercel.
2. Adicione as variáveis de ambiente:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
3. O deploy será feito automaticamente usando o `vercel.json` incluso.
